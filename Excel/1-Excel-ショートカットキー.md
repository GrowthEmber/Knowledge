# Excelショートカットキー

## 移動
| Win | Mac | 内容 |
| --- | --- | --- |
| Ctrl + N | ⌘ + N | 新規ブック |
| Ctrl + O | ⌘ + O | 開く |
| Ctrl + S | ⌘ + S | 保存 |
| F12 | ⌘ + Shift + S | 名前を付けて保存 |
| Ctrl + P | ⌘ + P | 印刷 |
| Ctrl + Z | ⌘ + Z | 元に戻す |
| Ctrl + Y | ⌘ + Shift + Z | やり直し |
| Ctrl + C | ⌘ + C | コピー |
| Ctrl + X | ⌘ + X | 切り取り |
| Ctrl + V | ⌘ + V | 貼り付け |
