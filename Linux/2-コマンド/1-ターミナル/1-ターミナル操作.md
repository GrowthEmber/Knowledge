# ターミナル操作

## カーソル移動
| キー | 内容 |
| --- | --- |
| 左右矢印キー | カーソル移動 |
| 上下矢印キー | コマンド履歴 |
| Ctrl + b | 一つ前に移動 |
| Ctrl + f | 一つ後ろに移動 |
| Ctrl + a | 行頭に移動 |
| Ctrl + e | 行末に移動 |
| Esc → b | 単語単位で一つ前に移動 |
| Esc → f | 単語単位で一つ後ろに移動 |

## 入力削除
| キー | 内容 |
| --- | --- |
| Ctrl + h | 後方一文字削除 |
| Ctrl + d | 前方一文字削除 |
| Ctrl + w | 一単語削除 |

## カット・ヤンク
| キー | 内容 |
| --- | --- |
| Ctrl + u | カーソル位置から行頭まで削除 |
| Ctrl + k | カーソル位置から行末まで削除 |
| Ctrl + y | 最後に削除した内容を挿入 |

## 履歴
| キー | 内容 |
| --- | --- |
| Ctrl + p | 前のコマンド履歴 |
| Ctrl + n | 次のコマンド履歴 |
| Ctrl + r | インクリメンタル検索 |

#### インクリメンタル検索の動作
| キー | 内容 |
| --- | --- |
| Ctrl + r | 一つ前の履歴 |
| Ctrl + g | 破棄 |
| Enter | 実行 |
| Esc | コマンドラインに戻る |

## その他
| キー | 内容 |
| --- | --- |
| Ctrl + c | 実行中のコマンドを強制終了 |
| Ctrl + l | 画面をクリアする |
| Ctrl + t | 一つ前とカーソルの文字を入れ替える
| Ctrl + m | Enter |
| Ctrl + j | Enter |
| Ctrl + o | Enter |
