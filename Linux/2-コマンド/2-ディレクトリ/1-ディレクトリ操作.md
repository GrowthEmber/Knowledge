# ディレクトリ操作

## 現在のディレクトリ表示
- コマンド: `pwd`

## 移動
- コマンド：`cd`
  - 別ディレクトリへ移動：`cd dir1/dir2/dir3`

## ディレクトリ作成
- コマンド: `mkdir`
  - 複数ディレクトリ作成可能：`mkdir dir1 dir2 dir3`

| オプション | 説明 |
|---|---|
| -p | 中間ディレクトリ作成 |
| -m | パーミッション指定 |
| -v | 作成内容表示 |

## 空ディレクトリ削除
- コマンド: `rmdir`
  - 空ディレクトリのみ削除
  - 複数ディレクトリ削除可能：`rmdir dir1 dir2 dir3`

## 移動・リネーム
- コマンド: `mv`
  - ディレクトリ名を変更（リネーム）：`mv old_dir new_dir`
  - 別ディレクトリ移動：`mv dir1 /home/user/`
  - 複数ディレクトリをまとめて移動：`mv dir1 dir2 dir3 /backup/`

| オプション | 説明 |
|---|---|
| -n | 上書きしない |
| -i | 上書き前に確認 |
| -f | 強制上書き |
| -v | 内容表示 |

## 削除
- コマンド: `rm`
  - 確認しながら削除（推奨）：`rm -i file.txt`
  - ディレクトリ削除（中身含む）：`rm -r dir/`
  - 複数ファイル削除：`rm file1.txt file2.txt`
  - 正規表現
    - logファイル全削除：`rm *.log`
    - test1.txt, test2.txt など：`rm test?.txt`

**ゴミ箱に入らないため、復元不可**
| オプション | 説明 |
|---|---|
| -i | 削除前に確認 |
| -f | 強制削除 |
| -r / -R | ディレクトリを再帰的に削除 |
| -v | 削除内容表示 |
| --preserve-root | `/` を誤削除しない |

## コピー
- コマンド: `cp`
  - アーカイブ：`cp -a src/ dst/`
    - `-a`は以下のセット
      - `-r`：再帰
      - `-p`：権限・所有者・時刻を保持
      - `-d`：シンボリックリンクをそのままコピー
  - ディレクトリコピー：`cp -r src/ dst/`
    - `-r`（recursive）が必須
    - これを忘れると 何も起きない or エラー
  - 複数ファイルを別ディレクトリにコピー：`cp a.txt b.txt dir/`

| オプション | 説明 |
|---|---|
| -r | ディレクトリ内を再帰的にコピー |
| -a | アーカイブ（属性保持 + 再帰） |
| -p | 権限・所有者・時刻を保持 |
| -i | 上書き前に確認 |
| -n | 上書きしない |
| -u | 新しいものだけコピー（差分コピー） |
| -f | 強制上書き |
| -v | 処理内容を表示 |
| -l | ハードリンク作成（容量節約） |
| -s | シンボリックリンク作成（設定切替） |
