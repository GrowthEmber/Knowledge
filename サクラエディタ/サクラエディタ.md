# サクラエディタ
- サクラエディタ (Sakura Editor) は、MS Windows 向けの無料・オープンソースの日本語テキストエディタ

# 正規表現

## 基本
| パターン | 意味 |
| --- | --- |
| . | 任意の1文字 |
| * | 直前の0回以上 |
| + | 直前の1回以上 |
| ? | 直前の0 or 1 |
| {n} {n,} {n,m} | 回数指定 |
| [] | 文字クラス |
| () | グループ化 |

## 文字クラス
| パターン | 意味 |
| --- | --- |
| [0-9] | 数字 |
| [a-zA-Z] | 英字 |
| \d | 数字 |
| \w | 英数字＋_ |
| \s | 空白（スペース/タブ等） |
| \D \W \S | 否定 |

## 位置指定
| パターン | 意味 |
| --- | --- |
| ^ | 行頭 |
| $ | 行末 |
| \b | 単語境界 |

<br>

# 実務で使う例
- 空白削除<br>　`^\s*$`
- 日付フォーマット変換
  - 置換前：`(\d{4})-(\d{2})-(\d{2})`
  - 置換後：`$1/$2/$3`<br>→ 2026-01-18 → 2026/01/18
- エラー行検索（大小無視）<br>　`(error|exception)`
- ログから不要な行削除
  - 置換前：`.*error.*\n`
  - 置換後：（空文字）
- ディレクトリからファイル名抽出
  - 置換前：`^.*[\\/]`
  - 置換後：（空文字）
- ファイル名から拡張子削除
  - 置換前：`\..*`
  - 置換後：（空文字）
