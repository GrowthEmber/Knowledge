# Teamsショートカットキー

## 全般
| キー | 内容 |
| --- | --- |
| Ctrl + E | 検索バーに移動 |
| Ctrl + / | クイックコマンドを開く |
| Ctrl + G | チームを表示 |
| Ctrl + 1 | アクティビティを表示 |
| Ctrl + 2 | チャットを表示 |
| Ctrl + 3 | チームを表示 |
| Ctrl + 4 | カレンダーを表示 |
| Ctrl + 5 | 通話を表示 |
| Ctrl + 6 | ファイルを表示 |
| Ctrl + N | 新しいチャットを開始 |
| Ctrl + Shift + N | 新しいチームを作成 |
| Ctrl + , | 設定を開く |
| Ctrl + . | その他オプションを開く |
| Ctrl + Enter | メッセージを送信 |

## ナビゲーション
| キー | 内容 |
| --- | --- |
| Tab | 項目間を移動 |
| Shift + Tab | 逆方向に項目間を移動 |
| Alt + Shift + 1 | 1つ目のピン留めされたチャットに移動 |
| Alt + Shift + 2 | 2つ目のピン留めされたチャットに移動 |
| Alt + Shift + 3 | 3つ目のピン留めされたチャットに移動 |
| Alt + Shift + 4 | 4つ目のピン留めされたチャットに移動 |
| Alt + Shift + 5 | 5つ目のピン留めされたチャットに移動 |
| Alt + Shift + 6 | 6つ目のピン留めされたチャットに移動 |
| Alt + Shift + 7 | 7つ目のピン留めされたチャットに移動 |
| Alt + Shift + 8 | 8つ目のピン留めされたチャットに移動 |
| Alt + Shift + 9 | 9つ目のピン留めされたチャットに移動 |
| Ctrl + Shift + B | アプリバーに移動 |

## チャットとメッセージ
| キー | 内容 |
| --- | --- |
| Ctrl + Shift + X | メッセージのフォーマットバーを展開/折りたたみ |
| Ctrl + Shift + C | 新しいチャネルを作成 |
| Ctrl + Shift + A | メッセージにアプリを添付 |
| Ctrl + Shift + F | チャットまたはチャネルを検索 |
| Ctrl + Shift + E | メッセージにファイルを添付 |
| Alt + Shift + C | 新しいチャネルを作成 |
| Alt + Shift + R | メッセージに返信 |
| Ctrl + Shift + R | リプライを開始 |
| Shift + Enter | メッセージ内で改行 |

## 通話と会議
| キー | 内容 |
| --- | --- |
| Ctrl + Shift + M | マイクのミュート/ミュート解除 |
| Ctrl + Shift + O | カメラのオン/オフ |
| Ctrl + Shift + U | 手を挙げる/手を下げる |
| Ctrl + Shift + K | 通話を終了 |
| Ctrl + Shift + L | ライブキャプションのオン/オフ |
| Ctrl + Shift + P | 画面共有のオン/オフ |
| Ctrl + Space | ミュート中に一時的にミュート解除 |
| Ctrl + Shift + H | 通話中の手を挙げる/手を下げる |
| Ctrl + Alt + D | カメラの背景ぼかしのオン/オフ |

## その他
| キー | 内容 |
| --- | --- |
| Ctrl + D | ダッシュボードに移動 |
| Ctrl + Shift + H | メンションされたメッセージの履歴を開く |
| Ctrl + Shift + M | 特定のチャットまたはチームの通知をミュート/ミュート解除 |
| Ctrl + Shift + Y | 絵文字を開く |
| Ctrl + Shift + H | 通話中の手を挙げる/手を下げる |
| Ctrl + Shift + K | 通話を終了 |
| Ctrl + Shift + L | ライブキャプションのオン/オフ |
